Cos'è CompactVD? ▲

CompactVD è un'utilità per ottimizzare la dimensione delle immagini disco dinamiche, basata sulle tabelle di allocazione bitmap dei file system conosciuti.
Puoi verificare se c'è spazio sprecato in un'immagine disco dinamica confrontando la dimensione del file immagine disco con lo spazio utilizzato dai file system all'interno dell'immagine. Per un'allocazione ottimale delle risorse, non dovrebbero essere molto diversi. Oppure trascina semplicemente un'immagine del disco in questa finestra:
I file system attualmente supportati sono:
  • New Technology File System (NTFS), per computer Windows
  • Hierarchical o Apple File System (HFS+ o APFS), per macOS
  • Extended File System (EXT), utilizzato nella maggior parte dei desktop Linux
Legge anche BTRFS o XFS (Linux), ma in questi casi la compressione non funziona.

Immagine del disco compatto al suo posto ▲

L'operazione compatta estrae blocchi di dati dalla fine dell'immagine del disco allo spazio che è stato rilevato come non utilizzato dai file system.
Per iniziare viene eseguita una scansione per rilevare i blocchi di dati che non sono utilizzati dai file system e la struttura dell'immagine del disco viene aggiornata contrassegnando tali blocchi come se non fossero mai stati utilizzati.
Quindi i blocchi con i dati effettivi vengono spostati dalla fine dell'immagine del disco allo spazio che è stato recuperato dall'operazione precedente.
Infine la struttura dell'immagine viene aggiornata per riflettere la nuova posizione dei blocchi spostati e la dimensione del file immagine viene tagliata. Gli aggiornamenti alla struttura dell'immagine vengono temporaneamente salvati sul supporto host e verranno scritti sull'immagine alla successiva esecuzione del programma, in caso di guasto hardware recuperabile.

Copia in una nuova immagine del disco ▲

L'operazione di copia inizia con una scansione per rilevare i blocchi di dati che non sono utilizzati dai file system. La nuova immagine viene creata ignorando quei blocchi e blocchi che altrimenti sarebbero completamente riempiti di zeri.
Il formato della nuova immagine disco può essere selezionato nella finestra di dialogo Copia immagine disco con una di queste estensioni di file:
  • VDI, Virtual Disk Image per Oracle VirtualBox
  • VMDK, Virtual Machine Disk per i prodotti VMware
  • VHD, Virtual Hard Disk per i prodotti Microsoft
  • RAW, per un'immagine disco completa
