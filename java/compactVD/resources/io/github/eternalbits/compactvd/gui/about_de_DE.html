Was ist CompactVD? ▲

CompactVD ist ein Dienstprogramm zum Optimieren der Größe von dynamischen Disk-Images, basierend auf den Bitmap-Zuordnungstabellen bekannter Dateisysteme.
Sie können überprüfen, ob in einem dynamischen Disk-Image verschwendeter Speicherplatz vorhanden ist, indem Sie die Größe der Disk-Image-Datei mit dem Speicherplatz vergleichen, der von Dateisystemen innerhalb des Images verwendet wird. Für eine optimale Ressourcenallokation sollten sie nicht sehr unterschiedlich sein. Oder ziehen Sie einfach ein Disk-Image in dieses Fenster:
Derzeit unterstützte Dateisysteme sind:
  • New Technology File System (NTFS) für Windows-Computer
  • Hierarchical oder Apple File System (HFS+ oder APFS) für macOS
  • Erweitertes Dateisystem (EXT), das in den meisten Linux-Desktops verwendet wird
Liest auch BTRFS oder XFS (Linux), aber in diesen Fällen funktioniert die Komprimierung nicht.

Compact Disk-Image vorhanden ▲

Die Komprimierungsoperation zieht Datenblöcke vom Ende des Disk-Images in den Speicherplatz, der von Dateisystemen als nicht verwendet erkannt wurde.
Zu Beginn wird ein Scan durchgeführt, um Datenblöcke zu erkennen, die nicht von Dateisystemen verwendet werden, und die Disk-Image-Struktur wird aktualisiert, wobei diese Blöcke markiert werden, als ob sie nie verwendet worden wären.
Dann werden Blöcke mit effektiven Daten vom Ende des Disk-Image in den Speicherplatz verschoben, der durch die vorherige Operation wiederhergestellt wurde.
Schließlich wird die Bildstruktur aktualisiert, um die neue Position der verschobenen Blöcke widerzuspiegeln, und die Bilddateigröße wird getrimmt. Aktualisierungen der Image-Struktur werden vorübergehend auf dem Host-Medium gespeichert und im Falle eines behebbaren Hardwarefehlers bei der nächsten Programmausführung in das Image geschrieben.

In neues Disk-Image kopieren ▲

Der Kopiervorgang beginnt mit einem Scan, um Datenblöcke zu erkennen, die nicht von Dateisystemen verwendet werden. Das neue Bild wird erzeugt, wobei diese Blöcke ignoriert werden, und Blöcke, die andernfalls vollständig mit Nullen gefüllt würden.
Das Format des neuen Disk-Images kann im Dialog Disk-Image kopieren mit einer dieser Dateierweiterungen ausgewählt werden:
  • VDI, Virtual Disk Image für Oracle VirtualBox
  • VMDK, Virtual Machine Disk für VMware-Produkte
  • VHD, Virtual Hard Disk für Microsoft-Produkte
  • RAW für ein vollständiges Disk-Image
