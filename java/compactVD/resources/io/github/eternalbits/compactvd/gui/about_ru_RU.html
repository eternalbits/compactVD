Что такое КомпактВД? ▲

CompactVD — это утилита для оптимизации размера динамических образов дисков на основе таблиц размещения растровых изображений известных файловых систем.
Вы можете проверить, есть ли неиспользуемое пространство в динамическом образе диска, сравнив размер файла образа диска с пространством, используемым файловыми системами внутри образа. Для оптимального распределения ресурсов они не должны сильно отличаться. Или просто перетащите образ диска в это окно:
В настоящее время поддерживаются следующие файловые системы:
  • Файловая система новой технологии (NTFS) для компьютеров с Windows
  • Иерархическая файловая система или файловая система Apple (HFS или APFS) для macOS
  • Расширенная файловая система (EXT), используемая в большинстве настольных компьютеров Linux.

Образ компакт-диска на месте ▲

Компактная операция извлекает блоки данных из конца образа диска в пространство, которое было обнаружено как неиспользуемое файловыми системами.
Для начала выполняется сканирование для обнаружения блоков данных, которые не используются файловыми системами, и структура образа диска обновляется, помечая эти блоки так, как если бы они никогда не использовались.
Затем блоки с действующими данными перемещаются из конца образа диска в пространство, восстановленное предыдущей операцией.
Наконец, структура изображения обновляется, чтобы отразить новое положение перемещенных блоков, а размер файла изображения обрезается. Обновления структуры образа временно сохраняются на носителе хоста и будут записаны в образ при следующем выполнении программы в случае устранимого сбоя оборудования.

Копировать в новый образ диска ▲

Операция копирования начинается со сканирования для обнаружения блоков данных, которые не используются файловыми системами. Новое изображение создается без учета этих блоков и блоков, которые в противном случае были бы полностью заполнены нулями.
Формат нового образа диска можно выбрать в диалоговом окне «Копировать образ диска» с одним из этих расширений файла:
  • VDI, образ виртуального диска для Oracle VirtualBox
  • VMDK, диск виртуальной машины для продуктов VMware
  • VHD, виртуальный жесткий диск для продуктов Microsoft
  • RAW, для полного образа диска
