¿Qué es CompactVD? ▲

CompactVD es una utilidad para optimizar el tamaño de las imágenes de disco dinámico, en función de las tablas de asignación de mapas de bits de los sistemas de archivos conocidos.
Puede verificar si hay espacio desperdiciado en una imagen de disco dinámica comparando el tamaño del archivo de imagen de disco con el espacio utilizado por los sistemas de archivos dentro de la imagen. Los sistemas de archivos compatibles actualmente son:
  • Sistema de archivos de nueva tecnología (NTFS), para computadoras con Windows
  • Sistema de archivos jerárquico o Apple (HFS+ o APFS), para macOS
  • Sistema de archivos extendido (EXT), utilizado en la mayoría de los escritorios de Linux
También lee BTRFS o XFS (Linux), pero en estos casos la compresión no funciona.

Imagen de disco compacto en su lugar ▲

La operación compacta extrae bloques de datos del final de la imagen del disco al espacio que los sistemas de archivos detectaron que no estaba en uso.
Para comenzar, se realiza un escaneo para detectar bloques de datos que no están en uso por los sistemas de archivos, y se actualiza la estructura de la imagen del disco marcando esos bloques como si nunca se hubieran utilizado.
Luego, los bloques con datos efectivos se mueven desde el final de la imagen del disco al espacio que fue recuperado por la operación anterior.
Finalmente, la estructura de la imagen se actualiza para reflejar la nueva posición de los bloques movidos y el tamaño del archivo de imagen se recorta. Las actualizaciones de la estructura de la imagen se guardan temporalmente en el medio del host y se escribirán en la imagen en la próxima ejecución del programa, en caso de una falla de hardware recuperable.

Copiar a nueva imagen de disco ▲

La operación de copia comienza con un escaneo para detectar bloques de datos que no están en uso por los sistemas de archivos. La nueva imagen se crea ignorando esos bloques y los bloques que de otro modo estarían completamente llenos de ceros.
El formato de la nueva imagen de disco se puede seleccionar en el cuadro de diálogo Copiar imagen de disco con una de estas extensiones de archivo:
  • VDI, Virtual Disk Image para Oracle VirtualBox
  • VMDK, Virtual Machine Disk para productos VMware
  • VHD, Virtual Hard Disk para productos de Microsoft
  • RAW, para una imagen de disco completa
