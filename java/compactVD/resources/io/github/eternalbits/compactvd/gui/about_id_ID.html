Apa itu CompactVD? ▲

CompactVD adalah utilitas untuk mengoptimalkan ukuran image disk dinamis, berdasarkan tabel alokasi bitmap dari sistem file yang dikenal.
Anda dapat memeriksa apakah ada ruang yang terbuang dalam image disk dinamis dengan membandingkan ukuran file image disk dengan ruang yang digunakan oleh sistem file di dalam image. Untuk alokasi sumber daya yang optimal, mereka seharusnya tidak jauh berbeda. Atau cukup seret gambar disk ke jendela ini:
Sistem file yang didukung saat ini adalah:
  • Sistem File Teknologi Baru (NTFS), untuk komputer Windows
  • Hierarchical atau Sistem File Apple (HFS+ atau APFS), untuk macOS
  • Extended File System (EXT), digunakan di sebagian besar desktop linux
Juga membaca BTRFS atau XFS (Linux), tetapi dalam kasus ini kompresi tidak berfungsi.

Gambar Disk Padat Di Tempat ▲

Operasi kompak menarik blok data dari ujung image disk ke ruang yang terdeteksi tidak digunakan oleh sistem file.
Untuk memulai pemindaian dilakukan untuk mendeteksi blok data yang tidak digunakan oleh sistem file, dan struktur gambar disk diperbarui menandai blok tersebut seolah-olah tidak pernah digunakan.
Kemudian blok dengan data efektif dipindahkan dari ujung gambar disk ke ruang yang telah dipulihkan oleh operasi sebelumnya.
Akhirnya struktur gambar diperbarui untuk mencerminkan posisi baru dari blok yang dipindahkan, dan ukuran file gambar dipangkas. Pembaruan struktur image disimpan sementara di media host dan akan ditulis ke image pada eksekusi program berikutnya, jika terjadi kegagalan perangkat keras yang dapat dipulihkan.

Salin ke Gambar Disk Baru ▲

Operasi penyalinan dimulai dengan pemindaian untuk mendeteksi blok data yang tidak digunakan oleh sistem file. Gambar baru dibuat mengabaikan blok-blok itu, dan blok-blok yang seharusnya diisi penuh dengan nol.
Format gambar disk baru dapat dipilih dalam dialog Salin Gambar Disk dengan salah satu ekstensi file berikut:
  • VDI, Gambar Disk Virtual untuk Oracle VirtualBox
  • VMDK, Disk Mesin Virtual untuk produk VMware
  • VHD, Hard Disk Virtual untuk produk Microsoft
  • RAW, untuk image disk yang lengkap
