O que é CompactVD? ▲

CompactVD é um utilitário para otimizar o tamanho de imagens de disco dinâmicas, com base nas tabelas de alocação de bitmap de sistemas de ficheiros conhecidos.
Você pode verificar se há espaço desperdiçado numa imagem de disco dinâmica comparando o tamanho do ficheiro de imagem de disco com o espaço usado pelos sistemas de ficheiros dentro da imagem. Para uma alocação ótima de recursos, eles não devem ser muito diferentes. Ou apenas arraste uma imagem de disco para esta janela:
Os sistemas de ficheiros atualmente suportados são:
  • New Technology File System (NTFS), para computadores Windows
  • Hierarchical ou Apple File System (HFS+ ou APFS), para macOS
  • Extended File System (EXT), usado na maioria dos desktops Linux
Também lê BTRFS ou XFS (Linux), mas nesses casos a compressão não funciona.

Imagem de Disco Compacto no Local ▲

A operação para compactar extrai blocos de dados do final da imagem do disco para o espaço que foi detectado como não sendo usado pelos sistemas de ficheiros.
Para iniciar é feita uma varredura para detectar blocos de dados que não estão em uso pelos sistemas de ficheiros, e a estrutura da imagem do disco é atualizada marcando esses blocos como se nunca tivessem sido usados.
Em seguida, os blocos com dados efetivos são movidos do final da imagem do disco para o espaço que foi recuperado pela operação anterior.
Por fim, a estrutura da imagem é atualizada para refletir a nova posição dos blocos movidos e o tamanho do ficheiro de imagem é reduzido. As atualizações na estrutura da imagem são salvas temporariamente na mídia do host e serão gravadas na imagem na próxima execução do programa, no caso de uma falha de hardware recuperável.

Copiar para Nova Imagem de Disco ▲

A operação de cópia começa com uma varredura para detectar blocos de dados que não estão em uso pelos sistemas de ficheiros. A nova imagem é criada ignorando esses blocos e blocos que de outra forma seriam completamente preenchidos com zeros.
O formato da nova imagem de disco pode ser selecionado na caixa de diálogo Copiar imagem de disco com uma destas extensões de ficheiro:
  • VDI, Virtual Disk Image para Oracle VirtualBox
  • VMDK, Virtual Machine Disk para produtos VMware
  • VHD, Virtual Hard Disk para produtos Microsoft
  • RAW, para uma imagem de disco completa
