什么是 CompactVD？ ▲

CompactVD 是一种基于已知文件系统的位图分配表来优化动态磁盘映像大小的实用程序。
您可以检查动态磁盘映像中是否存在浪费的空间，比较磁盘映像文件的大小与映像内文件系统使用的空间。 为了优化资源分配，它们不应该有很大的不同。 或者只是将磁盘映像拖到此窗口中：
目前支持的文件系统有：
  • 新技术文件系统 (NTFS)，适用于 Windows 计算机
  • 分层或 Apple 文件系统（HFS+ 或 APFS），适用于 macOS
  • 擴展文件系統 (EXT)，用於大多數 Linux 工作站
也读取 BTRFS 或 XFS (Linux)，但在这些情况下压缩不起作用。

光盘映像就位 ▲

压缩操作将数据块从磁盘映像的末尾拉到检测到未被文件系统使用的空间。
开始扫描以检测文件系统未使用的数据块，并更新磁盘映像结构，将这些块标记为从未使用过。
然后将具有有效数据的块从磁盘映像的末尾移动到之前操作回收的空间。
最后更新图像结构以反映移动块的新位置，并修剪图像文件大小。 图像结构的更新临时保存在主机介质上，并在下一次程序执行时写入图像，以防出现可恢复的硬件故障。

复制到新磁盘映像 ▲

复制操作从扫描开始，以检测文件系统未使用的数据块。 创建新图像时会忽略那些块，否则这些块将完全被零填充。
可以在“复制磁盘映像”对话框中使用以下文件扩展名选择新磁盘映像的格式：
  • VDI，Oracle VirtualBox 的虚拟磁盘映像
  • VMDK，VMware 产品的虚拟机磁盘
  • VHD，Microsoft 产品的虚拟硬盘
  • RAW，用于完整的磁盘映像
